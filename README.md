# RASPBERRY HOME SERVER
My docker environment 
## content

### host monitor 
host and containers stats monitor

current containers:
- grafana
- prometheus
- node-exporter
- cadvisor (custom build)

### media server
tv series, movies and media player

current containers:
- plex
- radarr
- sonarr
- bazarr
- jackett
- qbittorrent

### samba
SMB storage server

current containers:
- samba (custom build)
